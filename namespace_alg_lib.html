Computes the index ind that will partition the container arr

Returns index ind such that A[ind] \( \ge \) A[i] for \( startIndex \le i < ind \) and A[ind] \( \ge \) A[i] for \( ind < i \le endIndex \)

Parameters
TThe type of the container
startIndexThe starting index
endIndexThe ending index
Returns
Partitions arr into arr[startIndex...ind-1] and arr[ind+1...endIndex] and returns the value of ind. Note that this function will modify the container arr
