template<typename T >
void AlgLib::countingSort ( T &  arr)

Performs a counting sort on the container.

Sorts the container arr in linear time

Parameters
TThe type of the container e.g. std::vector<int> that stores ints
arrThe container to be sorted.

Performs the sort in O(n + k) where n is the number of elements and k is max_element - min_element.
