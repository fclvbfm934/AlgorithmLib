Performs a counting sort on the container.

Sorts the container arr in linear time, with the stipulation that arr must be a container of ints

Template Parameters
TThe type of the container e.g. std::vector<int> that stores ints
Parameters
arrThe container to be sorted.
startWhere to begin sorting. Defaults to 0.
lastThe index to stop sorting at. Note that last will not be included in the sort. Defaults to arr.size().

This will sort arr[start....(last-1)]. Performs the sort in O(n + k) where n is the number of elements and k is max_element - min_element.
